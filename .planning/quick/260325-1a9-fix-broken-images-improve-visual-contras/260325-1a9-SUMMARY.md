# Quick Task 260325-1a9: Fix broken images, improve visual contrast, enlarge hero logo

## Summary

Fixed critical rendering issues preventing content from being visible on the site.

## Issues Found & Fixed

### 1. Lazy Load + Scroll Reveal Race Condition (Critical)
- **Problem:** Images inside `.reveal` elements (opacity: 0) never triggered browser lazy loading, so they stayed blank even after reveal
- **Fix:** Added preload observer that switches `loading="lazy"` to `loading="eager"` 200px before viewport entry. Used double-rAF for `is-visible` class toggle to ensure CSS transition fires properly.

### 2. Brew Card Transition Conflict
- **Problem:** `.brew-card` had `transition: transform 0.4s` that conflicted with `.reveal` transition, preventing opacity animation
- **Fix:** Moved hover transition to `.brew-card.is-visible` so it only activates after reveal completes

### 3. Visual Contrast Too Low
- **Problem:** All sections blended into void black, cards invisible against dark backgrounds
- **Fixes:**
  - Alt section background: `#0D0D0D` → `#111111`
  - Added wood-brown top borders between sections
  - Brew cards: brighter bg (#141414), stronger gold borders, drop shadow
  - Gallery images: stronger borders and shadow
  - Rune dividers: added gold glow filter

### 4. Hero Logo Too Small
- **Fix:** Increased from `clamp(280px, 60vmin, 600px)` to `clamp(320px, 70vmin, 700px)` with triple-layer gold glow
